# 📊 Lending Portfolio Analysis

End-to-end financial risk and lending analysis using Python.

## 🔍 Project Overview
This project analyzes **1.04 Million+ loan records** to identify:
- Market demand trends
- Borrower risk scores
- Borrower demographics

## 🛠️ Tools & Technologies
- Python
- Pandas
- Seaborn
- Matplotlib
- Jupyter Notebook

## 📈 Key Insights
- Identified high-risk borrower segments
- Analyzed lending patterns across demographics
- Built risk scoring framework from raw data
